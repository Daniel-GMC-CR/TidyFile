# Política de Privacidad de TidyFile

**Última actualización: 14 de mayo de 2026**

## Información recopilada
**TidyFile NO recopila absolutamente ninguna información personal.**

## Funcionamiento offline
La aplicación funciona completamente offline. No envía datos a internet excepto para:
- Verificar si hay actualizaciones disponibles (consulta la API pública de GitHub)
- Descargar actualizaciones (solo si el usuario hace clic en "Actualizar")

## Datos almacenados
La aplicación solo almacena:
- Archivo version.txt: contiene la versión actual
- Archivos de configuración/reglas (solo si el usuario los crea)

## Sin telemetría
No se recopilan estadísticas de uso, no hay analytics, no hay tracking.
