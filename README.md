[Descargar última versión](https://github.com/Daniel-GMC-CR/TidyFile/releases/latest/download/TidyFile.exe)

### Opción 2

## ⚠️ Nota sobre seguridad de Windows
Si Windows muestra una advertencia de seguridad:
1. Haz clic en **"Más información"**
2. Luego en **"Ejecutar de todos modos"**

## 🚀 Características
- Organiza archivos por extensión automáticamente
- Totalmente offline - no envía datos a internet
- Solo verifica actualizaciones en GitHub
- Portable - no requiere instalación

## 📋 Sistema de actualizaciones
La app se actualiza automáticamente cuando hay una nueva versión disponible.
