TidyFile es una aplicación de escritorio enfocada en automatizar tareas avanzadas de manejo de archivos, procesamiento multimedia y reorganización documental desde una interfaz optimizada para trabajo masivo e individual.

El proyecto está diseñado como una plataforma modular en constante expansión, permitiendo integrar nuevas herramientas y sistemas conforme evolucionan las necesidades del flujo de trabajo.

---

# Características principales

## Organización automática de archivos

Permite separar archivos dentro de carpetas según su tipo o extensión.

Ejemplos:
- Imágenes
- Videos
- Audio
- Documentos
- Archivos comprimidos
- Otros formatos personalizados

Compatible con:
- Carpetas individuales
- Procesamiento recursivo en subcarpetas

---

## Renombrado avanzado de archivos

Permite renombrar archivos masivamente utilizando distintos criterios:

- Orden natural
- Fecha de modificación
- Fecha de creación

Diseñado para mantener secuencias consistentes en grandes cantidades de archivos.

---

## Compresión de video

Herramienta para reducir tamaño de videos manteniendo compatibilidad y automatizando procesamiento masivo.

Pensado para:
- almacenamiento,
- envío,
- respaldo,
- reorganización multimedia.

---

## Fusión de videos

Permite combinar múltiples videos automáticamente en un único archivo final.

Compatible con:
- procesamiento por lotes,
- secuencias ordenadas,
- múltiples carpetas.

---

## Transcripción de audio y video

Extrae y transcribe automáticamente contenido hablado desde:
- videos,
- audios,
- grabaciones.

Útil para:
- documentación,
- subtítulos,
- indexación,
- respaldo textual.

---

## Gestión y sobrescritura de metadatos

Permite:
- eliminar metadatos,
- sobrescribir información,
- limpiar rastros de archivos multimedia.

Compatible con:
- imágenes,
- videos.

---

## Reorganización avanzada de PDFs

Incluye herramientas para:
- dividir PDFs por rangos,
- extraer páginas,
- mover rangos internos,
- generar nuevos documentos.

También incorpora un mini visor optimizado de referencia visual para navegación rápida dentro del PDF.

---

## Generación de evidencias y documentación visual

Permite generar:
- PDFs a partir de carpetas con imágenes,
- evidencias visuales de código fuente,
- documentación automática basada en archivos.

Especialmente útil para:
- reportes,
- respaldos,
- entregas académicas,
- documentación técnica.

---

# Procesamiento masivo

La mayoría de herramientas pueden ejecutarse:
- sobre un único archivo,
- sobre carpetas completas,
- o recursivamente sobre subcarpetas.

El objetivo principal es reducir trabajo repetitivo y acelerar operaciones complejas de manejo de archivos.

---

# Sistema de versiones

TidyFile utiliza un sistema de versionado de tres niveles:

```text
X.Y.Z
```

Donde:

- `X` = Cambios estructurales grandes
- `Y` = Cambios intermedios importantes
- `Z` = Cambios leves, ajustes o mejoras menores

## Ejemplos

### Cambio leve

```text
1.4.7 → 1.4.8
```

### Cambio intermedio importante

```text
1.4.8 → 1.5.0
```

### Cambio estructural grande

```text
1.5.3 → 2.0.0
```

## Importante

El incremento de un nivel superior NO elimina el historial de versiones anteriores.

Por ejemplo:

- La versión `2.0.0` indica que existieron múltiples cambios y etapas previas dentro de la serie `1.x.x`.
- La numeración funciona como un registro evolutivo acumulativo del proyecto.
- Cada nivel representa una nueva etapa dentro de la continuidad histórica de la aplicación.

Esto permite:
- mantener trazabilidad completa,
- identificar etapas importantes del desarrollo,
- distinguir entre mejoras menores y cambios estructurales.
